API Overview

The AUTOPAY API allows merchants to initiate M-Pesa STK Push payments, verify transaction status, receive webhook notifications and manage payment collections securely.

Base URL

https://autopay-backend-1y64.onrender.com/api

Authentication

Every request must include your Secret API Key. 

Authorization: Bearer sk_live_xxxxxxxxxxxxx

POST /stk-push

{
  "amount":100,
  "phone":"254712345678",
  "accountReference":"ORDER1001",
  "description":"Payment",
  "callbackUrl":"https://yourdomain.com/webhook"
}

POST /check-status

{
  "checkoutRequestId":"ws_CO_xxxxxxxxx"
}

Webhook Response

{
 "status":"SUCCESS",
 "amount":100,
 "phone":"254712345678",
 "reference":"ORDER1001"
}

Error Codes

Code Description
400 Invalid Request
401 Unauthorized
404 Transaction Not Found
500 Internal Server Error

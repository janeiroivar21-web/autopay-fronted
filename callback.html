Configure your payment callback URL

Callback URL

AUTOPAY will send payment notifications to this URL whenever a transaction is completed successfully.

Must start with https://
Must accept POST requests
Receives JSON payload
Example Callback Payload 
{
  "success": true,
  "status": "SUCCESS",
  "amount": 100,
  "phone": "254712345678",
  "checkout_request_id": "ws_CO_123456789",
  "merchant_request_id": "29115-34620561-1",
  "transaction_id": "TXN123456"
}
